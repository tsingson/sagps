SAGPS (白雪回归线)

一个帮阿尔茨海默病（老年痴呆症）走失老人回家的 GPS 开源研究项目。

Dementia Search / Silver Alert (research) project, Search and Rescue GPS system






愿天下所有老人健康、清醒。希望大家永远用不到这个项目。

发起初衷

我是发起人 tsingson。我家里有轻度老年痴呆的老人，偶尔会迷路走失。在三四线城市或县城，现成的gps定位器, 智能手表或 Apple AirTag 并不好用（要么没信号，要么没 iPhone 帮传定位，要么老人嫌烦不肯戴）。所以，我决定自己折腾一个方案。

---

🛡️ 安全与隐私（绝不侵犯隐私）

因为管地理位置的法律很严，所以本项目不卖产品，也不提供公共定位服务。我们只给技术方案：

1. 自己买设备：家里备几个微型定位器缝在老人衣服里。
2. 自己搭服务器：数据存在自己能控制的地方，完全不对外公开，严格限制只能家里人查。
3. 数据不保留：(自己拥有的)服务端只存最近 24 小时的轨迹，最多 30 个家属能看。

---

🎯 我们的目标

- 对使用的家庭：提供一个藏得深、能防水、续航能撑 5 天的超小定位器，附带一键安装的查轨迹网页或小程序。
- 对想帮忙的技术人员：电路图、定位器代码（C 语言）、后台代码（Go 语言）全部开源，完全公开。
- 对精通硬件的朋友:  给有需要的家庭, 使用电路图进行定制. (我们提供固件, 及服务端程序等等)

---

🚦 目前的技术进展 (2026/07/08)

为了省电、降成本，我做过几轮车载和步行测试，目前推翻了之前的方案，确定了新的方向：

- 硬件：试了 u-blox m10 的超小 GPS 模块和中移的 4G 芯片（ML307R-DL），做了几个STM32和ESP32的样机。
- 软件调整：放弃了 ESP32 + FreeRTOS，换成了 STM32 + Zephyr RTOS 系统。因为 ESP32 太费电了，撑不到 5 天；而且换成 Zephyr 系统后，以后换其他芯片写代码更灵活。
