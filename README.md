# sagps
Dementia Search / Silver Alert  (research) project, Search and Rescue GPS system // 失忆老人寻回, 搜救 GPS 研究项目 ( 简称: 白银寻回)

## 0 简述
本项目成立于2026/05/28

这是一个早期的个人研究项目, 方向主要是针对(老年痴呆症 3级以内)的失忆老人走失后寻回, 作一些 GPS 定位相关的技术辅助, 及相关协助研究.

由于隐私相关的法律约束, 本项目只讨论一个可能性:

1. 个人携带GPS 定位微型设备(  10个以内充电备用, 两个在用)
2. 家庭( 老人的监护人) 自主部署互联网服务节点,该节点只为一个家庭的成员提供老人 GPS 位置/轨迹查询, 完全不对公众公开, 只在一个家庭内安全/限制使用
3. 本研究项目只提供技术方案, 及技术咨询, 不提供具体实施及相关 GPS 定位及轨迹查询等服务

## 2 当前进展

2026/07/08 进展:

1. 整体需求与技术架构作了初步研究, 有初稿. 服务器部分接口作了原型开发并验证, 其中, RTK 协议相关作了模拟测试, 因成本过高, 暂时放弃
2. gps 微型定位器 prototype 已经作了初步研究, 分别开发了基于 STM32 / ESP32 的几款 gps 定位原型, 并初步测试达到 gps 定位, 4g 上报.....等基础功能
   1. gps模块, 试验了 u-blox m10  等4个尺寸比较小的型号
   2. 4g 型号主要试验了 ML307R-DL 为主的几个小型开发板
   3. MCU 试了 STM32 的两款,  ESP32 试了 3款
   4. 当然, 用 ssd1306 小显示屏, 配置电池, gps 进行了车载/步行/小电驴...行进测试
3. 定位器废弃 platformio + freeRTOS + ESP-IDF 及 ESP32 作为主控 MCU 方案, 转采用 STM32  + zephyr 方案, 原因有
   1. ESP32 功耗管理达不到要求
   2. 4G 芯片/ GPS 芯片没有确定型号, 还在一个一个考量功耗控制与成本...要换
   3. 将来 gps 小型化, 也可能有变化的型号, 所以, 嵌入式代码要有一定灵活性
